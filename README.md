# In Brief
A demonstration test for Hotels.com using VueJs 2.0 and ExpressJs

# Stack
+ VueJs 2.0
+ Vue Cli
+ ExpressJs
+ Axios
+ Bootstrap
+ SCSS
+ Webpack

# Note
Fake data is supplied via server.js

# Installation
+ Download repository
+ npm install
+ run node server.js
+ open new terminal and then npm run start
